Contacts


Contacts module is all about the contact details of Landowners, Organization, Supplier and Contractors responsible for SNPA PRoW features. This module facilitates users with following functionalities:

  • View Contact Details
  • Add New Contact
  • Add Existing Contact
  • Delete Contact

  1. Display Contact Details


This functionality allows user to view the details of Contractor/Supplier/Organization/Landowner related to a selected feature. To view contact details:


  1. Select a SNPA feature (Path/Furniture/Surface/Access Land/Issue).
  2. Click on ‘Detailed Info’ tool from result panel.
  3. Click on Link tool of feature (Path Links/Furniture Links/Surface Links/Access Land Links/ Issue Links/Job Links) from the detailed information of feature.
  4. Application opens a new tab window according to type of feature (Path Links/Furniture Links/Surface Links/Access Land Links/ Issue Links/Job Links).
  5. Go to Contact accordion, application displays all the list of related contacts in tabular format (if exists).
  6. Click on Contact ID of a record, application displays the contact details on left panel in Contact tab.

  1. Add New Contact


This functionality allows user to add a new contact to selected feature. To add new contact follow the below steps:


  1. On Contact accordion of feature link tab, click on ‘Add New Contact’ button.
  2. Application displays Contact form in left panel.
  3. Select Contact Type and enter all the contact details.
  4. After filling all the information, click on ‘Save’ button.
  5. Contact details are saved to server and application displays message “Data Successfully Saved”.

  1. Add Existing Contact


This functionality allows user to add contact details to selected feature from the records available in SNPA database. To add an existing contact:


  1. On Contact accordion of feature link tab, click on ‘Add Existing Contact’ button.
  2. Application displays Contact form with a drop-down box Name and two buttons ‘Cancel’ and ‘Save’.

Fig.2 Adding Existing Contact


  1. Select name from the available list of contacts.
  2. Application displays all details of selected person.
  3. Click on ‘Save’ button, application displays message “Data Successfully Saved”.
  4. Click on ‘Cancel’ button, application closes the tab without adding the contact.

  1. Delete Contact


Delete functionality allows user to remove a contact details associated with selected feature. In Path Links/Furniture Links/Surface Links/Access Land Links/ Issue Links/Job Links under Contact accordion if any contact details exist, user can remove this contact details by following steps below:


  1. Click on Delete icon in Action column of record.
  2. Application displays a confirmation message to remove contact details.
  3. Click Ok, application displays another message of successful deletion.
  4. Click Cancel to discard the delete process.
