Text Style


Text style functionality allows users to label a specific attribute value on the features of active layer on the map canvas. User can label any field’s value on the features with different font style, colour and size.


  1. To label features, click on ‘Text Style’ tool bar. A tab will be shown to user on the left panel.
  2. The tab has three parts ‘Column’, ‘Font’ and ‘Halo’.
  3. In column part, the tab has a drop down list box which contains all the field names of the active layer.
  4. Font part has a drop down list box ‘Name’, a colour box ‘Colour’ and a spin text box ‘Size’ to increase and decrease the text size.
  5. And the third part halo has a spin text box to increase and decrease the halo radius around the text.
  6. A colour box ‘Fill’ and a slider ‘Opacity’. Below of this a button ‘Apply’ is also there.

To apply Text Style:


  1. To add label to a layer feature, first make that layer active.
  2. Click on ‘Text Style’ tool. Application will open ‘Text Style’ tab. Select the field in ‘Column’.
  3. In font part select the font type in ‘Name’ list box.
  4. In ‘Colour’ box user has to specify font colour. To select a font colour, click on the colour box, a colour picker will be shown to user. User can pick the colour by the cursor or by specifying the RGB/HSB/hexadecimal value of the colour.
  5. Then enter the font size in ‘Size’ spin box.
  6. Now in halo part, specify the radius of halo around a character of label.
  7. Select the halo colour and set the opacity level by using ‘Opacity’ slider.
  8. After doing all the settings, click on ‘Apply’ button. Now the features of active layer will be labelled with the specified attribute value.
  9. To remove labels, click on ‘Clear’ tool on toolbar.
