Job


Job module includes functionalities like view and edits job details, create job for issue and view contact and issue details associated with job. In this module user is allowed to upload a risk assessment document for a job.


  1. View Job Information


  1. To view job information user has to go to Work Commitment tab. From Open Issues section or Outstanding Jobs section user can view job details.
  2. In Open issues part, click on Create/View Job tool if Job does not exist for issue a new job form is opened otherwise the jobs details will be shown in left panel as Job tab in read only format.
  3. In outstanding job part click on Job ID and job details will be shown in left panel as Job tab read only format.
  4. User can view job details from the Issue part of feature (Path Links/Furniture Links/Surface Links/Access Land Links).
  5. On Issue part of feature link click on Create/View Job icon. If job is created for the issue, application displays job details in left panel in Job tab in read only format.

Fig.1 Job Details


  1. Edit Job Information


  1. To edit Job details click on Edit button. Application changes attribute format to editable with two buttons Cancel and Save.
  2. Edit required fields and click on ‘Save’ button. Application displays message for successful submission of changes.
  3. Click Cancel button to discard changes without saving changes.

  1. Job Links


In job links user can view and delete contacts associated with Job. And user can add contact to job. User can also view the issue detail for which job is created. To go to job links, click on Job Links tool   present on Job tab. Application opens a new tab Job Links in main window with two accordions: Contact (default open) and Issue.


Fig.2 Job Links


  1. Contact

In contact part user can view contact details associated with the job. User can add a new contact or add an existing contact to the job.


  1. To go to contact part, click on contact accordion. All contact details (if exists) will be shown in tabular format.
  2. To remove a contact, click on Delete icon in last column. Application displays confirmation message.
  3. Click Ok to Remove or click Cancel to discard.
  4. To add a new contact, click on Add New Contact button. Fill details of the contact in the form opened in left panel and click on Save.
  5. To add an existing contact, click on Add Existing Contact. Select contact from drop-down and click on Save button.
  6. Click on Go to Parent button on top of accordion, application displays the Job details  for which link is opened.

  1. Issue

In Issue part user can see parent issue of the job. Click on Issue accordion, application displays parent issue details in tabular format. Click on Issue ID, application opens issue information in left panel.


  1. Create Jobs


Job can be created from Open Issues in Work Commitment dashboard. And also from Issue part of features link (Path Links/Furniture Links/Surface Links/Access Land Links).

From Work Commitment in Open Issue accordion or from Path Links/Furniture Links/Surface Links/Access Land Links tab in Issue accordion, click on Create/View Job icon on last column.

If job is not created for selected issue, application opens Job tab with attribute fields. Follow the below steps to create new job:


  1. Enter all the field details (excluding auto filled fields).
  2. Click on ‘Save’ button. Application displays a message for successful creation of job.
  3. Click Cancel button to exit from job creation.

  1. Print Risk Assessment Document


In this user is allowed to upload risk assessment document to Job. The steps are described below:


  1. User clicks on Print Risk Assessment button, a new window will open with Risk Assessment template (which will be an external URL to RoW application). User can save the risk assessment document on his/her local machine.

 

Fig.2 Risk Attachment Box


  1. In Risk Assessment box, user selects Risk Assessment required option as Yes. Application expands the box to provide option to upload risk assessment document.  
  2. User enters Risk Assessment Reference and enters Risk Assessment documents location path or browse to its location with the help of Browse button.
  3. Click on Upload button. Application uploads the document and shows it in the document list.
  4. Select Assessment Completed as Yes.
  5. Risk assessment data will be saved with the Job.
  6. If user selects Yes for Assessment Completed without attaching risk assessment document, application shows a message that risk assessment document needs to be uploaded.
  7. User can remove a Risk Assessment document by click on Remove from Action column.
  8. Application displays a confirmation message. Click Ok to remove or click Cancel to discard remove process.
  9. If user selects No option for Risk Assessment Required, application shrink risk assessment box.

  1. Attach/Remove File


This functionality allows users to add files (up to 2MB) to the job. File could be of text type, image and other type.

For this application displays an attachment box with the list of already attached files. Attachment box consists of a text box ‘Description’, Browse button as File name and Attach button.


To add an attachment:

  1. Enter attachment description.
  2. Enter file path or click on Browse button to add file name from any specific location.
  3. Click on Attach button.
  4. On successful attachment, attached file is displayed in Attached Files list.

To remove an attachment:

  1. Click on the remove icon after the file name in Attached Files list on Job tab.
  2. Application displays a confirmation message to remove attachment.
  3. Click Ok to remove or Click Cancel to discard.
