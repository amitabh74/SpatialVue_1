Query Builder


Query Builder functionality allows users to query features according to the attributes values with the help of some operators. To query on layer, user have to zoom to at least max scale limit of layer i.e. layer should be visible on map.


  1. Click on the query builder tool, application opens a new tab ‘Query’ in left panel.
  2. This tab consists of two buttons: ‘Help’ and ‘Close’ in upper right corner,
  3. Below of them one drop down list box ‘Select Layer’ is there.
  4. Then it has three drop-down list boxes: ‘Fields’, ‘Operators’ and ‘Values’.
  5. Below to Values list box ‘See More’ link is there. The Values list box can have a list of 100 values at a time so to add more values to it one has to click on the see more link. It adds hundred more values for the selected attributes. On every click on that link it adds 100 mor values until all attribute values are not added.
  6. A drop down box ‘Load Query’ is there. This list box consists of the list queries saved by the user earlier. User can select a query from the list.
  7. Query tab has a text box ‘Where Clause’ in which user can see the query and the selected fields, operators and field’s values will be updated here automatically while selecting from  respective list available above.
  8. Above this text box a link ‘Clear’ is there to clear all the selected details.  
  9. Below to where clause text box a link ‘Save Query’ is there to save a query of user for future use.
  10. As user clicks on this link it expands with two text boxes: ‘Query Name’ and ‘Query Description’.
  11. At bottom, query tab has two buttons: ‘Save Query’ and ‘Run Query’. ‘Save Query’ button is used with the ‘Save Query’ link to save the query details. And ‘Run Query’ button is used to run the query which is shown in where clause.

Note: ‘Not’ operator should be used before the field name (for example: NOT objected=10, it will select the result which has not object as 10).


  1. Run Query  


To simply run a query:


  1. Select layer from the drop- down list on which query is desired.
  2. Select field from ‘Fields’ list box.  
  3. Select operator from ‘Operators’ list box, also select the field value from ‘Values’ drop-down list. User can create a query in which more than one field and more than an operator is used.
  4. User can see the whole query in ‘Where Clause’ text box.
  5. If the query is all correct then click on ‘Run Query’ button.
  6. Application will pops-up the result panel with queried features’ details on it.
  7. Now user can clear the where clause text box by clicking on clear link.



  1. Save Query


A query can be saved for future use. To save a query:


  1. Select layer, fields, operators and fields’ values.
  2. Click on ‘Save Query’ link.
  3. Enter query name in ‘Query Name’ text box and enter description of query in ‘Description’ text box.
  4. Click on ‘Save Query’ button. Application will shows a pop-up message on successfully saving the query to the server.



  1. Load Query


Load query allows user to load a saved query. To run a Saved query:


  1. Select layer for which query is saved.
  2. Select saved query from the drop-down list from ‘Load Query’ list box, application shows query name in Load Query drop-down and query in where clause.
  3. Click on ‘Run Query’ button. The results will be shown in results panel.
