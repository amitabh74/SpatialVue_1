Display/Edit Furniture Information


In this functionality user will be allowed to view any furniture information, and according to his role and rights, user will be able to edit furniture information. Also user can create issue on furniture.


  1. Display Furniture Information


In this functionality, user is allowed to view detailed information of selected furniture. For this user has to make furniture layer as active layer.


  1. User selects Furniture by Search/ Query Builder/ Select Feature/ Select by Rectangle/Select by Polygon. Selected feature/features should be displayed in result panel.
  2. Now select the required record from result panel and click on ‘Detailed Info’ tool on top of result panel.
  3. Application opens new tab ‘Furniture’ in left panel with the detailed information of selected furniture in read only format.

The Furniture information includes attributes as shown in below figures:


Fig.1 Furniture Information


This tab also includes


  1. Button : Edit
  2. Links: ‘Previous’ and ‘Next’
  3. Tools: View Furniture Links, Map It, Edit Furniture, Create Issue
  4. Attachment box with the list of already attached documents (if any) as shown in figure above.

Attach Files: In the attachment box, user is allowed to attach any type of file (up to 2mb) like test file or image file.To attach a file to furniture,


  1. Enter the description.
  2. Enter the file path or browse to the location of file.
  3. Click on Attach button.
  4. On successful attachment document will be displayed in the documents list.
  5. To remove an attachment, click on the remove icon in Action column. Application displays conformation message “Are You Sure you want to delete”.
  6. Click Ok and the document will be removed from list.
  7. Click Cancel to discard.
  8. To open a pre-attached document click on that, attached document will be opened.  

  1. Edit Furniture information    


  1. To edit furniture information, click on ‘Edit’ button in Furniture tab. Application changes the attributes to editable form.
  2. After doing the required changes in attribute fields, click on ‘Save’ button.
  3. Application saves the changes by creating the old records as historical details and the changes will be saved as current details of selected furniture.
  4. On saving the changes application displays message as “Data saved”.
  5. Click on Cancel button, application discard the changes and the attributes are changed to read only format.

  1. Display Historical details


This functionality allows users to view historical survey details of selected furniture as Historical Details.


  1. To view historical details of selected furniture, click on ‘Previous’ tool present on Furniture tab.
  2. On clicking ‘Previous’ tool application shows the historical details previous of current details.
  3. After first historical details, if user clicks on ‘Previous’ tool, application shows the message as “No Historical Records”.
  4. Now to view later version of the furniture details, click on ‘Next’ button. As user reaches to the latest version, application displays message as “No More Records”.

  1. Furniture Links


This tool allows user to view issues and contacts related to selected furniture. In this section user can see the parent path information of the selected furniture.


  1. Click on ‘Furniture Links’ icon on Furniture tab. Application opens a new tab ‘Furniture Links’ on map window.
  2. This window consists of three accordions: Contact, Issue and Path Details. Contact accordion is default open accordion with a button ‘Go to Parent’ on top.

Fig.2 Furniture Links


  1. Click on ‘Go to Parent’ button again shows the furniture details on left panel if furniture details tab is closed.
  2. To go any of the accordions, just click on that accordion.

These accordions are described below:


  1. Issues

  1. As user clicks on Issue accordion, application expands the accordion with the list of all open issues (if exists) in tabular format for selected furniture. This accordion has two buttons ‘Open Issues’ and ‘Historical Issues’.

Fig.3 Issue accordion


  1. To view the old issues on selected furniture, user clicks on ‘Historical Issues’ button. Application will display list of all issues those are resolved for selected furniture. If no historical issue exists for selected furniture, application shows a message “No historical issues are associated with selected Furniture”.
  2. To go to the open issues list again, click on Open Issues button.
  3. Click on ID of an issue to view its details.
  4. In the tabular list of the open issues, in the last column ‘Action’ table have two icons: Create/View Job and Create/View Legal.
  5. On clicking ‘Create/View job’ application opens a form in left panel as Job with  job details if already created or as a new form to create job for that issue if not created.
  6. Similarly if user clicks on ‘Create/View Legal’ icon, application opens a form in left panel as Legal with legal details if already created or a blank form to create a new legal.

  1. Contact

Contact part contains the list of contacts of SNPA contractors related to selected furniture. Here user is allowed to add new contact to Furniture and select a contact to add to furniture from the contractor’s existing list.


  1. As user clicks on a Contact accordion, application expands contact tab with the tabular list of contacts related to selected furniture.
  2. To view detailed information of a contact, click on ContactID of a record.
  3. In the last column of the table a tool ‘Delete’ exist to delete the contact record.
  4. To add a new contact, click on Add New Contact button or to add a contact from existing users list click on Add Existing Contact button.

  1. Path Details

In this section of furniture link, user is allowed to view the parent path information.


  1. Click on Path Details option, application expands the accordion with details of parent path on which that furniture exists.
  2. Click on RoWID of parent path. Application displays detailed information of the parent path in left panel.

Fig.4 Path Details of furniture


  1. Zoom to Selected Furniture on Map


With this functionality user is allowed to zoom to selected furniture from the detailed information of the furniture.

Click on ‘Locate on Map’ tool on Furniture tab. Application zooms to the selected furniture on map canvas.


  1. Edit Furniture


This functionality allows user to edit furniture like move furniture, delete furniture and edit furniture details without creating a historical detail. Click on the Edit Furniture tool, application opens Editing tab in left panel with furniture layer selected as editable layer.


  1. Create Issue on Furniture


User is allowed to mark an issue on Furniture under this functionality. To create an issue over Furniture:


  1. Click on ‘Create Issue’ button. Application opens editing tab in left panel with editing functionalities.
  2. Click on Create link in editing tab. Application expands Create link.
  3. Select point tool from Create part.
  4. Click on map over selected furniture at required location. If user clicks at a location other than selected furniture application displays message “No Furniture found at this location”.
  5. On clicking over furniture, application expands Edit Attribute link with all the issue details.
  6. Enter the values for all the fields (except some auto filled not editable fields). Click on ‘Apply’ button. Application validates the fields and displays message “Click on Save button to save the Issue”.
  7. Click ‘Ok’ and save the issue by clicking on ‘Save’ button in general editing tools. Application displays the issue on furniture.
