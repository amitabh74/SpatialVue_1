Info  


Info tool is used to see the attributes of features by clicking on them. To view information:


  1. Click on the Info tool, which enables Info tool.
  2. Now click on a desired feature. Application opens a new tab ‘Info’ in left panel.
  3. If multiple features exist at that location, application shows all features attributes in layer manager in collapsed form. By default first feature attributes are displayed in expanded form and it is higlighhted on map as shown in figure below. Fields name are displayed in left and respective values in right side.
  4. To view infomration of other feature shown in Info tab, click on name of that feature. Info tab expands its information and that feature will be highlighted on map.
