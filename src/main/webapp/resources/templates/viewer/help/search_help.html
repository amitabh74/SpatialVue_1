Search


Search functionality facilitates users to search features according to their attribute values. It provides options to search features under a specific boundary and there is also options to search features with attributes having similar value. Click on Search tool, application opens Search tab in left panel.To search a feature:


  1. Enter the value in the search text box.  
  2. Select the layer form the drop-down list and also select the fields of selected layer from the drop-down list.
  3. In the drop-down list there is an option available as ‘All’. If user selects ‘All’ for the fields then application will search for the results in all the fields of selected layer.
  4. If user does not selects any of the options from ‘Extent Boundary’ and ‘Type Options’ and clicks on ‘Search’ button. Application will search the result in entire layer with ‘Exact’ option as default.
  5. If entered value matches with attribute values, application will pop-up the result panel with the searched features’ attributes.
  6. Now For the same search user can specify type option as ‘Similar’ (Note: Similar option does not work for numeric values)
  7. If type option is chosen as ‘Similar’, application will show all the results which are similar to the entered value.

Users can also search features within a defined boundary:


  1. Click on ‘Search’ and specify the attribute value in Search text box.
  2. Select layer and field click on ‘Extent Boundary’ link. Two tools Polygon and Rectangle will be shown to user.
  3. Click on Polygon tool draw a polygon on the map canvas.
  4. Click on the type options link and specify type as ‘Exact’ or ‘Similar’.
  5. Then click on ‘Search’ button. If entered value matches with some of the features’ attribute value of specified layer and field inside the drawn polygon, application will pops-up the result panel with the details of these features.
  6. If the entered value does match with any features attribute of the specified layer and field, application shows a pop-up message of no record found. Click ‘Ok’ to exit from the message.
  7. Similarly user can search features inside an extent boundary by drawing a rectangle on map canvas using ‘Rectangle’ tool.
