Surveys


Under survey functionality user can re-assign a survey to some other user.


    Fig.1 Re-assigning Survey


  1. Select new user from drop- down list and change next survey date (if required).
  2. Click on ‘Save’ button, application displays message “Survey has been assigned successfully”.
  3. An admin user can see Backlog surveys (those survey by date has been passed) by clicking on Backlog button.
  4. Application displays all backlogged surveys list in tabular format with reassign button.
  5. User can see path details of any of these surveys by clicking on RoW ID of survey.
  6. To again go to pending surveys list, click on Pending button.
